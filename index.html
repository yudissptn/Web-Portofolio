Caesar Chiper

Input text to encode with certain rules. In this practice using caesar chiper ROT13. Which is shifting letter by 13 places Input form can be used to create encode text or to decode text Start with convert input string to uppercase, then check with regular expression is it whitespace or not. If yes concate the value and move to next value. If it is not a whitespace convert string to charcode. After getting the charcode value, add the ROT13 rules by adding 13 and check is it greater than last character charcode (Z = 77), if yes, loop back to charcode 'A'.
